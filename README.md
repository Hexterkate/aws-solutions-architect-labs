# AWS Solutions Architect Labs

Hands-on labs designed to implement and validate secure, scalable, and highly available architectures while studying for the AWS Certified Solutions Architect - Associate certification.

## 🏗️ Labs Overview

| Lab | AWS Service | Core Focus | Status |
| :--- | :--- | :--- | :--- |
| **Lab 01** | [IAM](./labs/iam) | Multi-tier security & Principle of Least Privilege (PoLP) | 🚧 In Progress |
| **Lab 02** | [S3](./labs/s3) | Data persistence, lifecycle management & encryption | 📅 Planned |
| **Lab 03** | [EC2](./labs/ec2) | Compute scaling, secure remote access & networking | 📅 Planned |

---

## 🛠️ Tech Stack & Tools
- **Cloud Provider:** Amazon Web Services (AWS)
- **Infrastructure as Code (IaC):** Terraform (Planned for automated labs)
- **Local Env:** Linux (CachyOS) & GitHub CLI

## 🚀 How to Use This Repo
Each folder inside the `labs/` directory contains its own specific architecture diagram, documentation, and implementation scripts. Navigate to a specific lab folder to see the complete hands-on guide.
